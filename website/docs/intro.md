OTerminus is a local, safety-first terminal assistant. It turns natural-language requests into a
single proposed shell action, previews that action, and keeps explicit confirmation at the center of
the execution flow.

This Docusaurus site is a temporary scaffold while the project migrates documentation from MkDocs.
The live documentation source remains the existing MkDocs `docs/` tree until the migration and
deployment switch are completed. Follow the project on
[GitHub](https://github.com/PooriaT/oterminus).
